1️⃣ What is the difference between var, let, and const?

-> var is function-scoped while let and const are block-scoped
   var and let can be re-assigned but const cannot

2️⃣ What is the spread operator (...)?

-> it can copy array or objects and without changing individual elements it can merge it to new ones

3️⃣ What is the difference between map(), filter(), and forEach()?

-> map()= transform elements and return new array
   filter()= select elements and return new array
   forEach()= loop through elements but doesn't return anything(undefined)

4️⃣ What is an arrow function?

-> a way to write function in js;which provides short syntax

5️⃣ What are template literals?

-> using backticks variables can be put inside a string easily
